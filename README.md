# Final Project: HIP Text Analysis

This repository contains my final project for text analysis using the Humanitarian Internship Program (HIP) dataset.

## Project Overview

This project uses computational text analysis to examine student reflective journals and supervisor interview data. The analysis explores recurring themes, skill-development patterns, challenges, and program learning opportunities.

## Repository Structure

- `data_clean_archival/` – cleaned or archived text data
- `metadata/` – metadata files used for analysis
- `outputs/` – figures, tables, and analysis outputs
- `Final Project- HIP Analysis.qmd` – main Quarto analysis file

## Methods

The project includes text cleaning, word frequency analysis, TF-IDF, dictionary-based categorization, named entity recognition, and exploratory topic modeling.

## Note

The original data are confidential and anonymized for research and educational purposes.
